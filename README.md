### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/MuadhGeorge/Personal-Portfolio-Website.git
   cd Personal-Portfolio-Website
   ```

2. **Install dependencies**
   ```bash
   pnpm install
   # or
   npm install
   # or
   yarn install
   ```

3. **Run the development server**
   ```bash
   pnpm dev
   # or
   npm run dev
   # or
   yarn dev
   ```
